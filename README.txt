

Single Computer InvertedIndex

We run this program in eclipse in windows.
You have to change the directory for reading data.
Put the invertedIndex.java and run.java into one java project. And run it. 
You will get the result in the directory you change.
And there will be the running time for the result.

Hadoop MapReduce indexing and ranking

Put the class as a jar file and start hadoop to use jar file to run the data we input.
InvertedIndex.java only do the inverted index part.
TFIDF_invertedIndex.java contains the code with TFIDF-ranking part.
You can run it and see the result using hdfs dfs -cat or hdfs dfs -get.


The specific details of how we implement it is in the report.
